# Resume Creator
Generates a resume (pdf) from the user's input.  
[👉LIVE DEMO](https://box-hill.github.io/resume-creator/#)

## 👍 Features
Github/Portfolio hyperlinks (optional)  
Custom Color Pickers  
Preset Themes  
Generate a sample resume  
Save as pdf  

## 🛠️ Made with
React.js  
Vanilla CSS  
npm
